# tech-devops 本地技能索引

本目录只服务 `tech-devops`。先按本索引选择技能，再完整读取对应 `SKILL.md`。禁止依赖 StartupPlanner Pro 外部的 Superpowers 源包。

| Skill | 用途 |
|---|---|
| [using-git-worktrees](./using-git-worktrees/SKILL.md) | 在授权且需要隔离时建立或识别 Git 工作区。 |
| [systematic-debugging](./systematic-debugging/SKILL.md) | 用跨组件证据、根因和单一假设处理故障。 |
| [verification-before-completion](./verification-before-completion/SKILL.md) | 对构建、测试、审查或交付结论生成新鲜证据。 |
| [finishing-a-development-branch](./finishing-a-development-branch/SKILL.md) | 仅在审核通过且得到授权后处理分支交付。 |

## 选择规则

- 只在触发条件成立时加载技能。
- 角色文件、Task Packet、授权范围和独立性规则优先约束技能。
- 不能验证的结论必须标记限制，不得用自报完成代替证据。
- 本目录是独立副本，源包更新不会自动覆盖。
