# tech-qa 本地技能索引

本目录只服务 `tech-qa`。先按本索引选择技能，再完整读取对应 `SKILL.md`。禁止依赖 StartupPlanner Pro 外部的 Superpowers 源包。

| Skill | 用途 |
|---|---|
| [test-driven-development](./test-driven-development/SKILL.md) | 为验收、缺陷复现和测试工具先证明测试有效。 |
| [systematic-debugging](./systematic-debugging/SKILL.md) | 用跨组件证据、根因和单一假设处理故障。 |
| [verification-before-completion](./verification-before-completion/SKILL.md) | 对构建、测试、审查或交付结论生成新鲜证据。 |

## 选择规则

- 只在触发条件成立时加载技能。
- 角色文件、Task Packet、授权范围和独立性规则优先约束技能。
- 不能验证的结论必须标记限制，不得用自报完成代替证据。
- 本目录是独立副本，源包更新不会自动覆盖。
