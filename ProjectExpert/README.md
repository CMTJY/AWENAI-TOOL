# ProjectExpert - AI项目专家部门

> 一个覆盖全行业的AI智能体团队，通过对话引导式交互，帮助用户快速了解、分析和规划任何领域的项目。

## 🎯 团队定位

AI项目专家部门是一个综合性的项目分析智能体团队，能够：

- 🔍 **轮询识别项目意图**：通过逐个问题的对话，精准识别用户想了解的项目
- 📍 **精准定位项目**：锁定用户真正要深入了解的具体项目
- 🌐 **全面检索信息**：跨行业、多维度地搜集项目相关信息
- 🏗️ **输出技术栈与知识库**：梳理项目技术体系，制定学习推进计划
- 📐 **需求分析与规划**：系统化的需求分析和可执行的项目推进方案
- 💡 **关键信息降智**：用大白话让任何人都能快速理解新项目

│

│   ├── 全域侦察兵        # 跨行业全面检索

│

## 🔄 协作流程

```
用户输入 → 需求引导 → 情报检索 → 规划分析 → 降智输出 → 最终交付
```

1. **需求引导部**：意图探针轮询提问 → 项目定位师锁定目标
2. **情报检索部**：全域侦察兵多维检索 → 知识图谱师结构化整理
3. **规划分析部**：技术架构师梳理技术栈 → 需求规划师制定方案
4. **降智输出部**：降智翻译官输出通俗总结
5. **输出总控**：整合所有成果，生成最终交付物

## 📋 输出交付物

每次项目分析完成后，输出以下交付物：

1. **📋 完整项目分析报告** — 包含项目概述、技术栈、知识库、需求分析、推进方案
2. **🗺️ 技术栈图谱** — 分层展示项目技术体系
3. **📚 学习推进计划** — 个性化学习路线和里程碑
4. **📐 需求分析文档** — 系统化的功能/非功能需求分析
5. **💡 降智总结** — 3分钟速读版，让任何人快速理解项目

## 🚀 使用方式

1. 在 Trae 中打开本项目
2. 直接描述你想了解的项目（越模糊越好，智能体会引导你）
3. 意图探针会逐个问题帮你明确需求
4. 团队会自动完成后续的分析和输出
